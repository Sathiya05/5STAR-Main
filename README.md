# 5 Star Event Management

A professional event management website featuring various decoration categories, galleries, and service details.

## Project Structure

The project follows a clean and organized structure for easy maintenance and deployment:

- `/` - Root directory containing HTML pages and the main data file.
- `assets/`
  - `images/` - All event and decoration photos.
  - `css/` - Stylesheets.
  - `js/` - JavaScript files.
- `decor-gallery-data.json` - Data file for the dynamic gallery.

## Features

- Dynamic galleries for different events (Weddings, Corporate, Birthdays, etc.)
- Responsive design
- Interactive contact forms
- Detailed service pages

## Local Development

To run the project locally, you can use any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then open `http://localhost:8000` in your browser.
